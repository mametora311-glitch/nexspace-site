サイト調整ガイド

色・余白・カードサイズ・ヒーロー高さなどを変数で一発調整。

1. 変えるのは /css/params.css だけ

以下の代表パラメータを変更するとサイト全体に反映されます。

色まわり

--nx-hero-start / --nx-hero-end     … ヒーローの上/下色--nx-grid                           … ヒーロー格子の濃さ(0〜0.1)--nx-bg / --nx-card / --nx-text     … ページ背景 / カード / 本文色--nx-muted / --nx-primary / --nx-border / --nx-shadow / --nx-radius

タイポ・横幅・余白

--container-w     … コンテナ最大幅（例: 1100px→1200px）--fz-hero/h2/h3   … 見出しサイズ--fz-body, --lh-body … 本文サイズ/行間--gap-top/bottom  … ヘッダー直下/フッター直前の全体余白

ヒーローの上下サイズ

--hero-pad-top / --hero-pad-bottom--hero-pad-small / --hero-pad-xsmall

例：高さを増やすなら --hero-pad-top:84px; --hero-pad-bottom:64px;

カードの大きさ・並び

--grid-gap          … カード同士の間隔--card-min-two      … 2列グリッドの最小幅（大きいほど1枚が広く）--card-min-three/four--card-pad, --card-min-h, --card-shadow

2. トップのProductsをヒーロー調にする

トップページの対象セクションに class="section hero-bg" を付与します。 色味は params.css の以下で調整します。

--hero-bg-card-bg / --hero-bg-border / --hero-bg-text / --hero-bg-muted / --hero-bg-btn

3. ページ別の微調整

/css/pages/*.css に追記します。例：

/css/pages/aiec-light.css.section .container > .card:first-child{ margin-top:6px; }

4. 差し替えリクエストのルール

今後、私からの差し替え指示はこの形式で出します：

  • ファイル指定：例）/css/base.css
  • 範囲指定：例）.hero{ の "padding:" 行〜次の "}" まで
  • 置換内容：置換後コードを完全提示（前後の数行を合わせて明示）

行番号は環境差でズレるため、基本は「開始トークン〜終了トークン」で範囲を特定します。
